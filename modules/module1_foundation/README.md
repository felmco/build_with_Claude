# Module 1: Foundation & Setup

## Overview
This module covers the fundamentals of working with Claude API, including understanding the available models, setting up your development environment, and making your first API call.

## Learning Objectives
By the end of this module, you will be able to:
- Understand the different Claude models and their use cases
- Set up your Python development environment with the Anthropic SDK
- Authenticate with the Claude API
- Make your first API call
- Understand basic API parameters and responses
