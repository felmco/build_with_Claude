# Module 2: Core API Features

## Overview
This module covers the core features of the Claude API including the Messages API, streaming responses, conversation management, and working with multimodal inputs.

## Learning Objectives
By the end of this module, you will be able to:
- Work with the Messages API effectively
- Implement streaming responses for better UX
- Manage multi-turn conversations
- Work with images and PDFs
- Handle files and documents
- Implement proper error handling and retries
