# 5.4 Quality Metrics

What defines "Good"?

## 1. Correctness
Does it answer the question factually?

## 2. Helpfulness
Did it solve the user's problem?

## 3. Safety
Did it refuse harmful instructions?

## 4. Tone
Is it polite/professional/on-brand?

## 5. Hallucination Rate
How often does it make things up?
