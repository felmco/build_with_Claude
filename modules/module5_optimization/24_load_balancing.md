# 5.6 Load Balancing

## Model Load Balancing
If you have multiple API keys (e.g., different organizations or fallbacks), you can load balance across them.

## Region Balancing
If using Bedrock/Vertex, you can route traffic to `us-east-1`, `us-west-2`, etc., to get higher limits.
