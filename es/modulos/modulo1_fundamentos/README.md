# Módulo 1: Fundamentos y Configuración

## Visión General
Este módulo cubre los fundamentos para trabajar con la API de Claude, incluyendo la comprensión de los modelos disponibles, la configuración de tu entorno de desarrollo y la realización de tu primera llamada a la API.

## Objetivos de Aprendizaje
Al final de este módulo, serás capaz de:
- Entender los diferentes modelos de Claude y sus casos de uso
- Configurar tu entorno de desarrollo Python con el SDK de Anthropic
- Autenticarte con la API de Claude
- Realizar tu primera llamada a la API
- Entender los parámetros básicos de la API y las respuestas

## Requisitos Previos
- Conocimiento básico de programación en Python
- Python 3.7 o superior instalado
- Un editor de texto o IDE (VS Code, PyCharm, etc.)
- Una cuenta de la API de Anthropic (regístrate en https://console.anthropic.com)
