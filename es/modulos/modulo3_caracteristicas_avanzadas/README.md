# Módulo 3: Características Avanzadas

## Visión General
Este módulo explora las características avanzadas de la API de Claude, incluyendo el uso de herramientas (function calling), almacenamiento en caché de prompts, procesamiento por lotes, capacidades de visión y pensamiento extendido (extended thinking).

## Objetivos de Aprendizaje
Al final de este módulo, serás capaz de:
- Implementar el uso de herramientas para integraciones externas
- Optimizar costes con almacenamiento en caché de prompts
- Procesar peticiones en lotes (batches)
- Trabajar con imágenes y contenido visual
- Usar pensamiento extendido para razonamiento complejo
- Integrar capacidades de Uso de Computadora (beta)
