# 5.6 Cumplimiento y Privacidad de Datos

## Política de Datos de Anthropic
- **API Estándar:** Los datos NO se usan para entrenamiento (revisa los términos más recientes).
- **Retención:** 30 días por defecto.

## GDPR / CCPA
Si un usuario pide ser borrado, debes borrar sus datos de tus registros.
- Claude no almacena datos de usuario a largo plazo, pero *tú* sí (en logs/DB).
