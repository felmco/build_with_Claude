# 5.1 Aprendizaje Few-Shot

El aprendizaje Few-shot implica proporcionar al modelo algunos ejemplos de pares "Entrada -> Salida Deseada".

## Zero-Shot vs Few-Shot

**Zero-Shot:**
> "Clasifica este correo: 'Mi reembolso llega tarde.' -> ?"

**Few-Shot:**
> "Clasifica este correo: '¡Me encanta este producto!' -> Positivo
> 'Esto está roto.' -> Negativo
> 'Mi reembolso llega tarde.' -> ?"

## ¿Cuándo usarlo?
- Tareas de clasificación.
- Requisitos de formato específicos.
- Imitar un tono o estilo específico.
