# Módulo 2: Características Principales de la API

## Visión General
Este módulo cubre las características principales de la API de Claude, incluyendo la API de Mensajes, respuestas en streaming, gestión de conversaciones y el trabajo con entradas multimodales.

## Objetivos de Aprendizaje
Al final de este módulo, serás capaz de:
- Trabajar con la API de Mensajes de manera efectiva
- Implementar respuestas en streaming para una mejor experiencia de usuario (UX)
- Gestionar conversaciones de múltiples turnos
- Trabajar con imágenes y PDFs
- Manejar archivos y documentos
- Implementar un manejo de errores adecuado y reintentos
