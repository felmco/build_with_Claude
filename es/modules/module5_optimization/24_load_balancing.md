# 5.6 Balanceo de Carga

## Balanceo de Carga de Modelos
Si tienes múltiples claves API (ej. diferentes organizaciones o fallbacks), puedes balancear la carga entre ellas.

## Balanceo de Región
Si usas Bedrock/Vertex, puedes enrutar tráfico a `us-east-1`, `us-west-2`, etc., para obtener límites más altos.
